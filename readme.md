# Perfumatch 
[Perfumatch](https://teekismikiss.github.io/perfumatch/)
**Perfumatch** es una herramienta diseñada específicamente para optimizar el trabajo de una vendedora independiente de perfumes en modelos de venta multinivel. 

La idea nació de una necesidad real: dejar de perder tiempo buscando en archivos PDF interminables y tediosos para encontrar la equivalencia perfecta.



##  Características
* **Búsqueda Instantánea:** Encuentra equivalencias por nombre de perfume original o marca .
* **Optimizado para Móvil:** Diseñado para usarse "en el campo" mientras hablas con un cliente.
* **Base de Datos Personalizada:** Creado a partir del PDF específico de equivalencias del vendedor.
* **Interfaz Limpia:** Sin distracciones, directo al grano.

## 🛠️ Cómo funciona
1.  Se extrajeron los datos del PDF original de equivalencias.
2.  Se estructuraron los datos para permitir búsquedas dinámicas.
3.  El motor de Perfumatch filtra los resultados en milisegundos.
